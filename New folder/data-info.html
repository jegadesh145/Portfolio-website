Quick Start Guide

Submit a Form

Go to your portfolio and fill out the contact form with your name, email, and message.

Data is Saved

The submission automatically saves to both Google Sheets and your browser's local storage.

Access Admin Panel

Visit admin.html and login with password: admin123

View & Export

See all submissions in a table, view details, or export to CSV (Excel) or JSON format.

Security Note

The admin panel is password protected with a default password admin123

To change this, edit the admin.html file and find:

const ADMIN_PASSWORD = "admin123";

Change "admin123" to your own secure password.
